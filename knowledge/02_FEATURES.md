# Фичи продукта (что есть, чего нет)

## @transkrib_smartcut_bot и @transkrib_plus_bot — РЕАЛЬНЫЕ возможности (оба бота phone-video)
- ✅ Принимает видеофайлы напрямую с устройства (до 20 МБ, форматы MP4, MOV, AVI и др.).
- ✅ Транскрипция речи через Groq Whisper с автоопределением языка.
- ✅ Форматирование текста с эмодзи-разметкой (🎬📌💡) через Claude.
- ✅ Нарезка видео на смысловые блоки (ffmpeg).
- ✅ 4 формата выдачи: текст, нарезка, SRT, нарезка+SRT.
- ❌ YouTube, VK, Rutube, ссылки — **не поддерживаются**. Только файлы с устройства.
- **CTA в постах: используй @transkrib_plus_bot** (основной бренд).

## ВНУТРЕННИЕ СИСТЕМЫ (для тем behind_scenes)
- Транскрипция: Groq Whisper. Используется whisper-large-v3-turbo для текста и whisper-large-v3 для SRT-субтитров. Обе модели поддерживают 99 языков с автоопределением.
- Анализ и нарезка: Anthropic Claude разбивает текст на смысловые блоки.
- Нарезка видео: FFmpeg с перекодировкой через libx264 (preset fast, crf 28) и aac (128 kbps). НЕ "без перекодировки" — это техническая неточность.
- Хостинг: Telegram-боты (lucid-youthfulness, transkrib-bot, transkrib-news-bot) хостятся на Railway. API backend для обработки видео (transkrib-api.onrender.com) хостится на Render. НЕ путать: "обработка на Render", "боты на Railway".
- Не упоминай Watchdog с 5 методами скачивания (yt-dlp, RapidAPI, Cobalt, Supadata, Telegram Direct) — это методы URL-flow, в phone-video пайплайне используется только Telegram Direct (bot.getFile).
